Observed vs True Lift

Your experiment result is not the truth — it's one noisy photograph of it

Every A/B test draws a random sample from two populations: control users and variant users. The observed conversion rates fluctuate around the true rates, and the observed lift fluctuates around the true lift.

Below, the two bell curves show the sampling distributions — the range of conversion rates you'd see across many experiments with this sample size. Each experiment draws one point from each curve.

Try these scenarios

  1. Properly powered: Set true lift = MDE (both 5%). Run 1,000 experiments. The sig rate should be close to your power setting.
  2. False positives: Set true lift to 0%. Run 1,000 experiments. About α% are significant — these are all false positives.
  3. Winner's curse preview: Set true lift to 1% with MDE at 5%. The few significant results show inflated observed lifts.
  4. Exaggeration gap: Compare the true lift line to the mean observed lift. When do they diverge?

Key Takeaways

A single experiment result is one noisy draw from a distribution. The sampling distributions show the range of possible outcomes — wider distributions (from smaller samples) mean more variable results. When you ship based on a single significant result, you're acting on one photograph, not the full album. Properly powering your experiment narrows the distributions and brings observed lifts closer to truth.
